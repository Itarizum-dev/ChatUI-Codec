# Stock Investment Advisor

このスキルは、株式の投資判断をサポートするための情報収集と分析を行います。

## When to use this skill
このスキルを使用するタイミング:
- ユーザーが特定の銘柄について投資判断を求めている
- 株式市場の動向や企業分析が必要な場合
- 投資リスクの評価を依頼された時

## How to use

### 1. 情報収集フェーズ
以下の情報をWeb検索ツールを使って収集します:

**必須情報:**
- 企業の基本情報（業種、事業内容、市場シェア）
- 最新の株価と過去1年間の価格推移
- 財務指標（PER, PBR, ROE, 自己資本比率、営業利益率）
- 最新の決算情報（売上高、営業利益、純利益の前年比）
- 配当利回りと配当性向
- 最近のニュース・プレスリリース（過去3ヶ月）

**推奨情報:**
- アナリストの目標株価とレーティング
- 競合他社との比較
- 業界全体のトレンド
- マクロ経済環境（金利、為替など）

### 2. 投資判断基準

以下の基準に基づいて総合的に評価します:

#### 財務健全性 (30%)
- **良好**: 自己資本比率 > 40%, ROE > 10%
- **注意**: 自己資本比率 < 30%, ROE < 5%
- **危険**: 自己資本比率 < 20%, 営業赤字

#### バリュエーション (25%)
- **割安**: PER < 業界平均 かつ PBR < 1.0
- **適正**: PER が業界平均±20%の範囲内
- **割高**: PER > 業界平均×1.5

#### 成長性 (25%)
- **高成長**: 売上高成長率 > 15%, 営業利益成長率 > 20%
- **安定成長**: 売上高成長率 5-15%
- **低成長**: 売上高成長率 < 5%

#### リスク要因 (20%)
- 業界の将来性と競争環境
- 規制リスク・地政学リスク
- 経営陣の信頼性
- 最近のネガティブニュースの有無

### 3. 推奨フォーマット

分析結果は以下の形式で提示します:

```
## 【銘柄名】投資分析レポート

### 基本情報
- 銘柄コード: XXXX
- 現在株価: X,XXX円
- 時価総額: XXX億円
- 業種: XXXX

### 財務指標
- PER: XX倍 (業界平均: XX倍)
- PBR: X.X倍
- ROE: XX%
- 自己資本比率: XX%
- 配当利回り: X.X%

### 評価サマリー
- 財務健全性: ⭐⭐⭐⭐☆ (4/5)
- バリュエーション: ⭐⭐⭐☆☆ (3/5)
- 成長性: ⭐⭐⭐⭐⭐ (5/5)
- リスク: ⭐⭐⭐☆☆ (3/5)

### 総合判断
**推奨: 買い / 中立 / 売り**

【理由】
- ポジティブ要因: ...
- ネガティブ要因: ...
- 投資期間: 短期 / 中期 / 長期

### 注意事項
この分析は情報提供を目的としており、投資を推奨するものではありません。
最終的な投資判断はご自身の責任で行ってください。
```

## Tools Required
- **Web検索ツール**: 企業情報、株価、ニュースの収集に使用
- **ファイルシステムツール**: 過去の分析結果の保存・参照（オプション）

## Example Usage

**User**: "トヨタ自動車(7203)の投資判断をお願いします"

**Action**:
1. Web検索で「トヨタ自動車 7203 株価」「トヨタ自動車 決算」「トヨタ自動車 ニュース」を検索
2. 財務指標を収集（Yahoo!ファイナンス、会社四季報など）
3. 上記の判断基準に基づいて評価
4. レポート形式で結果を提示

## References
- `references/investment-criteria.md`: 詳細な投資判断基準
- `references/industry-benchmarks.md`: 業種別の平均指標

## Notes
- 情報は常に最新のものを使用すること
- 複数の情報源からクロスチェックを行うこと
- リスク要因は必ず明記すること
- 法的な投資助言ではないことを明示すること
