# 投資判断基準 詳細

## 1. 財務健全性の評価基準

### 自己資本比率
- **優良**: 50%以上 → 財務基盤が非常に強固
- **良好**: 40-50% → 安定した財務状態
- **普通**: 30-40% → 一般的な水準
- **注意**: 20-30% → やや不安定
- **危険**: 20%未満 → 財務リスクが高い

### ROE (自己資本利益率)
- **優秀**: 15%以上 → 資本効率が非常に高い
- **良好**: 10-15% → 適切な収益性
- **普通**: 5-10% → 平均的な水準
- **低い**: 5%未満 → 収益性に課題

### 営業利益率
- **高収益**: 10%以上
- **標準**: 5-10%
- **低収益**: 5%未満

## 2. バリュエーション評価

### PER (株価収益率)
業種別の目安:
- IT・ハイテク: 20-30倍
- 製造業: 12-18倍
- 金融: 8-12倍
- 小売: 15-20倍

### PBR (株価純資産倍率)
- **割安**: 1.0倍未満
- **適正**: 1.0-2.0倍
- **割高**: 2.0倍以上

## 3. 成長性の評価

### 売上高成長率 (前年比)
- **高成長**: 15%以上
- **中成長**: 5-15%
- **低成長**: 0-5%
- **減収**: マイナス成長

### 営業利益成長率
- **優秀**: 20%以上
- **良好**: 10-20%
- **普通**: 0-10%
- **悪化**: マイナス成長

## 4. リスク評価チェックリスト

### 事業リスク
- [ ] 主力製品・サービスの競争力は維持されているか
- [ ] 新規事業の成長見込みはあるか
- [ ] 業界全体の成長性はどうか

### 財務リスク
- [ ] 有利子負債は適正水準か (D/Eレシオ < 1.0)
- [ ] キャッシュフローは健全か
- [ ] 配当は継続可能か

### 外部環境リスク
- [ ] 為替変動の影響は大きくないか
- [ ] 原材料価格の変動リスクは
- [ ] 規制変更のリスクは
- [ ] 地政学リスクは

## 5. 投資判断マトリックス

| 財務健全性 | バリュエーション | 成長性 | 総合判断 |
|---------|--------------|-------|---------|
| 高 | 割安 | 高 | **強気買い** |
| 高 | 適正 | 高 | **買い** |
| 高 | 割高 | 高 | 中立 |
| 中 | 割安 | 中 | 買い |
| 中 | 適正 | 中 | 中立 |
| 低 | - | - | **売り/様子見** |

## 6. 投資期間別の重視ポイント

### 短期投資 (3ヶ月以内)
- 最新のニュース・材料
- テクニカル指標
- 市場センチメント

### 中期投資 (6ヶ月〜1年)
- 四半期決算の推移
- 業績予想の達成度
- 業界トレンド

### 長期投資 (1年以上)
- ビジネスモデルの持続可能性
- 経営陣の質
- 競争優位性
- 配当成長性
